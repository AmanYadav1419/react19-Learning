## Annotations :-

- Annotations are used to `specify the data type` of `variable, parameter, function return value and other types of values`.
- Annotations helps developers catch errors `early in development` by allowing them to specify what types of values can be assigned to a given variable or passed to as an argument to a function.

## String Annotations 👇 :-

- Syntax :-

```ts
let myVar: type = Value;

// myVar is a variable name
// type is what Annotations or data type it will stores that is type
// Value is the value that being stored to the variable myVar.
```

- Example :-

```ts
let myName: string = "Aman Yadav";

myName = "Changed Name"
console.log(myName) // answer will be ➡️ Changed Name

// suppose if we try to store some different type value to the myName variable.
// This below line will gives the error as we have annoted the type for that variable is `string`✅ not `number` ❎
myName = 12

```
