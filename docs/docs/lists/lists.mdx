## Lists

- In React, you will render lists with some type of loop.
- The Javascript map() array method is generally the preferred method.

Here is the example of the Map method used in jsx ðŸ‘‡

```jsx
const ListExample = () => {
  const numbers = [1, 2, 3, 4, 5];
  return (
    <main>
      {numbers.map((number) => (
        <ul key={number}>
          <li>{number}</li>
        </ul>
      ))}
    </main>
  );
};

export default ListExample;
```

- remember the point each child in a list should have unique "key" prop.

Second Example of the Map method used in jsx ðŸ‘‡

- now we want to iterate all the user details in the userInfo array so,
  in this we are mapping an array of objects

```jsx
const ListExample1 = () => {
  const userInfo = [
    {
      username: "Aman",
      email: "aman@gmail.com",
      location: "India",
    },
    {
      username: "Yadav",
      email: "yadav@gmail.com",
      location: "Bharat",
    },
    {
      username: "Trion",
      email: "trion@gmail.com",
      location: "Hindustan",
    },
  ];

  return (
    <main>
      {userInfo.map((user) => (
        <ul key={user.email}>
          <li>{user.username}</li>
          <li>{user.location}</li>
          <li>{user.email}</li>
        </ul>
      ))}
    </main>
  );
};

export default ListExample1
```
