## useReducer :-

- useReducer is a hook that is similar to useState but it is designed for more complex state objects or state transitions that involes multiple subvalues.
- It allows you to manage state in a functional, immutable way.

- synatx :-

```jsx
const [state, dispatch] = useReducer(reducer, initialState);
```

- syntax explanation :-

- initialState :-
  - This is the starting value for the state when the component first renders.
  - e.g :- so if you are building a counter application you will pass something like a value 10 for the initial value of the counter.

- reducer :-
  - This is a function that describes how the state should change based on actions.
  - It takes the current state and an action as inputs , and returns a new state.

- state :-
  - This is the value we are destructuring from hook.
  - The current state value, which you can use in your component.

- dispatch :-
  - This is the value we are destructuring from hook.
  - A function you call to send actions to the reducer, which then updates the state.
